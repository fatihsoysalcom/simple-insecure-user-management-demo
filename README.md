# Simple Insecure User Management Demo

This example demonstrates a naive and insecure approach to building a user management system from scratch. It highlights the inherent complexities, missing security features (like proper password hashing, multi-factor authentication, and account recovery), and potential vulnerabilities. This illustrates the technical debt and security risks that the article advises against, making a strong case for specialized SaaS solutions.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Run from your terminal: `python main.py`
